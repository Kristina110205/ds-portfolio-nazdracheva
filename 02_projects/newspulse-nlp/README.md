# 📰 NewsPulse NLP

> Агрегатор новостей с анализом тональности и выявлением влияния на финансовые рынки

## 🎯 Цель проекта
Автоматический сбор новостей из открытых источников, анализ эмоциональной окраски и корреляция с движением рыночных индикаторов.

## 🚀 Статус
- [ ] Сбор данных (парсинг / API)
- [ ] Предобработка текста (токенизация, лемматизация)
- [ ] Анализ тональности (базовая модель)
- [ ] Визуализация результатов
- [ ] Деплой (веб-интерфейс / Telegram-бот)

## 📚 Технологии (план)
- Python 3.10+
- Pandas, NumPy — обработка данных
- BeautifulSoup / Scrapy / aiohttp — парсинг
- nltk / spacy / razdel — NLP для русского языка
- transformers (BERT) — продвинутый анализ
- Plotly / Streamlit — визуализация
